Customer Feedback System (React JS)
📌 Project Overview

The Customer Feedback System is a simple web application built using React JS that allows users to submit feedback and view it on a dashboard. The system collects information such as the user's name, email, feedback category, priority level, and description.

Submitted feedback is displayed dynamically in a Feedback Dashboard for easy viewing and management.

🚀 Features

Submit customer feedback through a form

Input validation for required fields

Select category and priority level

Display submitted feedback in a dashboard card layout

Dynamic rendering using React state

Clean and responsive UI

🛠 Technologies Used

React JS

JavaScript (ES6)

HTML5

CSS3

React Hooks (useState)

📋 Feedback Form Fields

The form collects the following information:

Name – User's name

Email – User's email address

Category – Type of feedback (e.g., Bug, Feature Request, Suggestion)

Priority – Importance level (Low, Medium, High)

Description – Detailed feedback message

🖥 How It Works

User fills the Customer Feedback Form.

After clicking Submit, the data is stored in React state.

The feedback appears in the Feedback Dashboard.

Each feedback is displayed as a card containing:

Name

Email

Category

Priority

Description

📸 UI Sections
1. Customer Feedback Form

Input fields for feedback details

Submit button to add feedback

2. Feedback Dashboard

Displays submitted feedback

Uses card-based UI
